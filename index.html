About the Development Process

This project is 100% open source, and I want to be open about how it was made:

Some parts were created with help from AI, but not because I had to — I already had working code from earlier projects. I used AI to help adjust these parts for this project, which saved me a lot of time.

When I switched to pywebview, many functions needed changes to work properly. Instead of rewriting everything by hand, I used AI to handle the repetitive parts so I could focus on the main work and make sure everything fit together well.

I have carefully checked every part of the code, and I take full responsibility for how the final version works. Some parts of the UI also use AI-generated elements to keep things looking consistent, which saved me from doing a lot of small, manual changes.

This is a free tool that I'm building in my free time. If you find any problems, feel free to DM me (.gekkefries1) — I really appreciate any feedback!
